# 🎲 Dice Roller App

A simple and interactive **Dice Roller Android App** built using modern Android development tools. This app simulates rolling a dice and displays a random result each time you tap the button.

---

## 🚀 Features

* 🎲 Roll a dice with a single tap
* 🔢 Generates random numbers (1–6)
* 🖼️ Displays dice face images dynamically
* ⚡ Smooth and responsive UI
* 📱 Beginner-friendly Android project

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **Framework:** Android SDK
* **UI:** Jetpack Compose / XML (whichever you used)
* **IDE:** Android Studio

## ⚙️ How It Works

1. User taps the **Roll Dice** button
2. App generates a random number between 1 and 6
3. Corresponding dice image is displayed
